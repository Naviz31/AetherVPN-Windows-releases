# AetherVPN Windows 0.1.3

Windows client stability update.

## Changes

- Removed invalid DNS detours from the sing-box 1.13+ DNS server configuration.
- Added a preflight `sing-box check` before starting the VPN core.
- Cleaned VPN core errors before showing them in the app.
- Limited the administrator hint to real TUN startup failures.
