# AetherVPN Windows Releases

Public downloads for the AetherVPN Windows client.

## Download

Download the latest installer from the repository Releases page:

https://github.com/Naviz31/AetherVPN-Windows-releases/releases/latest

Installer file:

`AetherVPN Setup 0.1.2.exe`

## Notes

This repository contains public release metadata only. The Windows installer is
published as a GitHub Release asset because it is larger than GitHub's normal
100 MB git file limit.
